# Recuperar VisionTetris

## Repositorio
https://github.com/DanielR850/VisionTetris

## Descargar
git clone https://github.com/DanielR850/VisionTetris.git

## Entrar al proyecto
cd VisionTetris

## Crear entorno virtual
python -m venv venv

## Activar entorno virtual
source venv/Scripts/activate

## Instalar dependencias
pip install -r requirements.txt

## Ejecutar
python main.py
