# Recuperar VisionTetris

Repositorio:
https://github.com/DanielR850/VisionTetris

## Descargar el proyecto

git clone https://github.com/DanielR850/VisionTetris.git

cd VisionTetris

## Crear entorno virtual

python -m venv venv

## Activar entorno virtual

venv\Scripts\activate

## Instalar dependencias

pip install -r requirements.txt

## Ejecutar

python main.py
