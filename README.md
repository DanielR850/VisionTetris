# VisionTetris

An interactive Tetris game controlled through real-time hand gestures using Python, OpenCV, MediaPipe and Pygame.

## Description

VisionTetris is a desktop game that reimagines classic Tetris by allowing the player to control the game using hand gestures detected through a webcam.

## Technologies

- Python
- OpenCV
- MediaPipe
- Pygame
- PyInstaller

## Features

- Real-time hand tracking
- Gesture-based controls
- Classic Tetris gameplay
- Webcam interaction
- Windows executable version
- Score, levels and next-piece preview

## Gestures

- Peace sign: move left
- Index finger: move right
- Fist: rotate piece
- Open hand: move piece down
